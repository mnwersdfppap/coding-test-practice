## 찬구의 코드
```java
import java.util.Scanner;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);

    int a = sc.nextInt();
    int b = a;
    int count = 0;

    while(true) {
      a = a % 10 * 10 + (a / 10 + a % 10) % 10;
      count++;
      if (a == b) {
        break;
      }
    }
    System.out.println(count);

    sc.close();
  }
}
```
