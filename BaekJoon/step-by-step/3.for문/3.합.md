## 찬구의 코드
```java
import java.util.Scanner;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);

    int n = sc.nextInt();
    int tmp = n;
    for(int i = 1; i <= tmp - 1; i++) {
      n += i;
    }

    System.out.println(n);
    sc.close();
  }
}
```
